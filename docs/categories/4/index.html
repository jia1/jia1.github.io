Adding my sites to Cloudflare

13 Sep 2020 @ 11.34 am
#software #domains #cloudflare
I added my domains to Cloudflare. Cloudflare offers many cool features which my domain providers do not offer. Well, they did not need to offer those features. In Cloudflare, I get to see web traffic analytics, set up DNSSEC, configure network transport options and/or optimisations, run speed tests on my site, configure caching, run JavaScript workers and even restrict image hotlinking. This list is non-exhaustive. There are more features in the Pro and Enterprise subscriptions.
